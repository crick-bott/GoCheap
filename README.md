# GoCheap 🤖✈️🚆🚌⛴️
A Telegram bot that helps you find the **cheapest tickets** for flights, trains, buses, and ships.

---

## ✨ Features (planned)
- 🔍 Search cheapest tickets by route & date  
- 🛫 Compare across flights, trains, buses, ferries  
- 💸 Price drop alerts  
- 📢 Daily deals feed  
- 🌍 Multi-currency support
